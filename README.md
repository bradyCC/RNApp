# ReactNativeApp

## 项目初始化
1. 安装：npm i -g react-native-cli
2. 项目初始化：ract-native init 项目名称

## 环境搭建
1. android
+ Node
+ React Native 命令行工具
+ Python
+ JDK
+ Android Studio

2. ios
+ Node
+ Watchman
+ React Native 命令行工具
+ Xcode
